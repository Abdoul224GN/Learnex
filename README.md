# Learnex

Bienvenue sur **Learnex** !  
Learnex est le site vitrine officiel d’une école spécialisée dans les formations en développement. Ce projet a pour vocation de présenter l’établissement, ses valeurs, ses programmes de formation, ainsi que les informations nécessaires pour toute personne souhaitant rejoindre ou travailler avec l’école.

## Description

- Présentation de l’école et de ses services
- Catalogue des formations en développement (web, mobile, etc.)
- Actualités, témoignages et réalisations des étudiants
- Informations pour les inscriptions et contacts
- Interface moderne et responsive

## Installation

Clonez le dépôt sur votre machine :

```bash
git clone https://github.com/Abdoul224GN/Learnex.git
cd Learnex
```

Installez les dépendances (adaptez la commande selon la technologie du projet, par exemple npm, yarn, ou pip) :

```bash
# Exemple pour un projet Node.js
npm install
# ou pour Yarn
yarn
```

## Lancer le projet

Exécutez le projet en local :

```bash
npm start
# ou
yarn start
```
